
  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
  ## technologies used 
   
    React 18 - Main frontend framework with hooks (useState, useEffect)
    TypeScript - Type-safe JavaScript development
    Vite - Modern build tool and development server
    HTML5 - Base markup structure
    Tailwind CSS V4 - Utility-first CSS framework with custom design tokens
    shadcn/ui - Pre-built accessible component library including
    Lucide React - Icon library for UI icons
    CSS Variables - Custom properties for theming (light/dark mode)
## Backend & Database
    Supabase - Backend-as-a-Service platform providing:
    PostgreSQL database
    Edge Functions
    Real-time subscriptions
    Authentication (ready for implementation)
    Storage (for file management)
    Deno - Runtime for Supabase Edge Functions
    Hono - Fast web framework for the server API
